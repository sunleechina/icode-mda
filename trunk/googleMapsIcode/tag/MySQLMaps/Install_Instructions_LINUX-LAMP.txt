-Installing Apache's httpd, PHP, MySQL on Fedora
-
yum -y install httpd php mysql mysql-server php-mysql mysql-workbench



Your web files will now be found in "/var/www/" 
- the document root.



You can work with your database using a gui by running:

mysql-workbench



run the workbench and then execute the SetupDBForICODE.sql



Paste:

index.html

icode-maps.js

icode_example_db_query.php

...to /var/www/html



Open a Web browser and goto:


http://localhost
